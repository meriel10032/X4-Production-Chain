# X4 Foundations Interactive Production Chain Map

An interactive production chain map for X4 Foundations v9.0.

- Multi-language support
- Visualizes upstream & downstream dependencies
- Auto-calculates required factory modules based on target output

Free to use, modify, and distribute. No attribution required.

## How to Download & Use

1. Go to [Releases](../../releases/latest)
2. Download the `.html` file
3. Open it in any browser — no installation needed
